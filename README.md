
## 9. RAG 구현
- 실습1. General Rag 구현


```text

파이썬 langchain 패키지를 이용해서 rag를 만들어줘
- 로컬 PC에 설치된 파이썬 패키지와 패키지 버전은 현재 폴더의 requirements.txt를 참고해 줘.
- flask 패키지를 이용해 웹페이지를 만들어줘
- user interface는 chatgpt와 유사하게 만들어줘
- 버튼, 창 스타일은 연한 청색, 회색 컬러로 산뜻하게 만들어줘 
- 왼쪽 메뉴창 상단에 파일 업로드 버튼표시 (여러개 파일 업로드) 
- pdf 파일만 업로드 할꺼야
- llm은 gpt-4o-mini 모델을 사용해야해. api-key는 나의 시스템 환경에 있으니 설정 안해도 돼
- 이파일을 PyPDFLoader 로 로딩해줘
- RecursiveCharacterTextSplitter로  500글자 분할, 100글자 오버랩
```

- 실습3. 계약서 검토 Rag 구현


```text

내가 만들려고 하는 웹 시스템은 아래와 같아

0. 제목 : LLM 기반 계약서 검토 웹

1. 개요
- 로컬 PC에 설치된 파이썬 패키지와 패키지 버전은 현재 폴더의 requirements.txt를 참고해 줘.
- 파이썬 langchain 0.3X 버전 패키지를 이용해서 LLM 기반 계약서 검토 웹 코드를 생성해줘
- 계약서 가이드라인이나, 약관을 업로드해서 rag를 구현하고, 사용자가 계약서를 업로드하면
  한 문장씩 읽고, rag에서 유사 문서를 검색후, 가이드라인이나 약관에 위배사항이 있는지 검토하고 계약서를 수정해서 출력하는 웹서비스를 구현하는 거야

2. 패키지 버전 및 환경
- flask 패키지를 이용해, llm은 gpt-4o-mini 모델을 사용해
- langchain 등 패키지 버전은 내 시스템에 설치된 ‘requirements.txt' 파일을 참조해줘
- api-key는 나의 시스템 환경에 있으니 설정 안해도 되
- 시스템 구현을 위해 추가적으로 필요한 패키지 버전은 알려줘
- 코드들은 이해하기 쉽게 주석을 추가해줘

3. 메뉴 구성
- user interface는 chatgpt와 유사하게 만들어줘 (왼쪽 : 메뉴, 오른쪽 : 채팅 및 결과 )
- 화면 구성은 연한 파랑색과, 연한 회색을 이용해 줘
- 왼쪽 메뉴는 'rag 파일 업로드' 버튼 과  '계약서 업로드' 버튼이 있어야 해
- '계약서 업로드가 완료되면 '계약서 검토' 버튼이 보여야 해
- 오른쪽 화면은 답변을 보여주고, 하단에는 일반적인 질문을 할 수 있는 질문창을 보이게 해줘
- rag 파일 업로드, 계약서 업로드시 답변 창에 진행율을 보여주고 완료 결과를 출력해줘 (tqdm 패키지 활용, 바 출력)

4. rag 구현
- 'rag 파일 업로드' 버튼을 누르면 pdf 파일을 업로드 할 수 있도록 구성해줘
- PyPDFLoader로 여러개의 파일을 업로드 할 수 있어야 해
- 업로드한 문서를 RecursiveCharacterTextSplitter로  30글자로 분할, 5글자로 오버랩해줘, separators는 ["\n", "\n\n"]
- text-embedding-3-small 로 임베딩하고 chromadb에 저장해

5 계약서 검토 구현
- '계약서 업로드' 버튼을 누르면 PyPDFLoader로 pdf 파일을 업로드 할 수 있게 해줘
- 계약서를 RecursiveCharacterTextSplitter로  30글자로 분할, 오버랩 하지마, separators는 ["\n", "\n\n"]
- 계약서 문장과 rag 문서를 비교하여 계약서 문장에 오류 또는 보완이 필요한 부분을 찾은 후 수정해줘
- 계약서 문장을 '[원문]' 이라고 표시후 출력하고 오류를 수정한 문장을 '[수정 문구]' 라고 표시한 후  진한 파란 글씨체로 원문의 바로 밑에 출력해줘 (7. 출력 예시 참조)
- 계약서 문장에 이상이 없다면 원문만 출력하면 됨
- 화면에 문장 하나씩 하나씩 출력해줘, 한꺼번에 출력되면 검토하는 동안 진행 여부를 확인할 수 없어

6. 기타
- 사용자가 일반적인 질문을 하게 되면 rag를 이용하지 말고 llm에 직접 요청해서 답변을 생성하면 되
- 만약 성능 향상을 위해 에이전트가 필요하다면 langgraph 패키지를 이용해서 에이전트를 구현해줘

7. 출력 예시
```
